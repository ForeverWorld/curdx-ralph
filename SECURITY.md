# Security Policy

## Supported Versions

Security fixes are provided for the latest `main` branch and latest tagged release.
Older snapshots are not guaranteed to receive backports.

## Reporting a Vulnerability

Do not report security issues in public GitHub issues.

Use one of these channels:

1. GitHub Security Advisories (preferred):  
   `https://github.com/ForeverWorld/curdx-ralph/security/advisories/new`
2. If advisory submission is unavailable, open a private maintainer contact request
   in Discussions and ask for a secure channel.

Please include:

- impact and attack path
- affected files/paths
- reproduction steps or PoC
- suggested mitigation (if known)

## Response Targets

- initial acknowledgement: within 72 hours
- triage update: within 7 days
- fix target:
  - critical/high: within 30 days
  - medium/low: within 90 days
